优化

既然差分之后就可以求出所有区间，为什么不直接往队列里面丢呢？

如差分序列2 1 0 0 0 0 -3，进行以下处理：

  1. 遇到正数222，直接丢222个下标111在存lll的队列中。

  2. 遇到正数111，直接丢111个下标222在存lll的队列中。

  3. 遇到负数-3，直接丢∣−3∣=3|-3|=3∣−3∣=3个下标7−1=67-1=67−1=6在存rrr的队列中。

  4. 输出即可。

时间复杂度O(n+m)O(n+m)O(n+m)。
