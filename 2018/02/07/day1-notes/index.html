错排问题

\(f_n\)

\(f_n=(n-1)(f_{n-1}+f_{n-2})\)

逆序对

\(f_{n,m}\)代表\(n\)个数产生\(m\)个逆序对的数量。

\(f_{n,m}=\sum_{k=1}^nf_{n-1,m-k+1}\)

前缀和优化

\(f_{n,m}=g_{i,m}-g_{i,m-n}\)

例题2 【IMPOSSIBLE】

例题3

二叉树：\(f_n=\sum_{i=0}^{n-1}f_if_{n-i-1}\binom {n-1} i\) 二叉树的子结构性质

有根树：\(f_n = \sum_{i=1}^{n-1}f_if_{n-i}\binom{n-2}{i-1}\) 去掉一棵子树还是一棵树

\(k\)叉树：f_{n,k}={i=1}^{n-1}f{i,1}f_{n-i,k-1}

\(g_{n,m}=\sum_{i=1}^{n-1}g_{n-i,m-1}f_i\)

\(f_i=\sum_{j=0}^kg_{i,j}\)

\(g_{n,m}\) 根\(m\)其它\(n\)

拆分数

\(f_{i, j} = f_{i, j-i} + f_{i-1, j} - i\)

\(n \le 10^5\)情况

\(g_{i,j}\) \(i\)个数，和为\(j\)。

\(i \le \sqrt n\)时：\(f_{i,j}=f_{i-1,j}+f_{i,j}-i\)

\(i \ge \sqrt n\)时：\(g_{i,j}=g_{i-1,j-1}+g_{i,j-i}\)

\(h_j = \sum_{i=0}^\sqrt ng_{i,j-i\sqrt n}\)

\(Ans = \sum_{A=0}^nf_{\sqrt n, A}h_{n-A}\)
