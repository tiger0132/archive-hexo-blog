【数学】Codeforces 872C - Maximum splitting

发表于 2018-02-06 | 分类于 数学

题意

传送门：Codeforces 872C

给你nnn个数，问你这些数最多可以写成多少个合数的和。n≤105,1≤x≤109n \le 10^5, 1 \le x \le 10^9n≤10​5​​,1≤x≤10​9​​

解法

考虑分解的策略。

首先，最小的444个合数分别是4,6,8,94, 6, 8, 94,6,8,9。

很明显，分解得到的合数越小越好。分类讨论：

  1. 当xxx是1,2,3,5,7,111, 2, 3, 5, 7, 111,2,3,5,7,11时无解，输出−1-1−1。
  2. 当xxx是偶数时，因为x>2x>2x>2，而且444和666可以组成所有大于222的偶数。所以只需要用444和666即可。
  3. 当xxx是奇数时，因为大于111111的奇数和999自身都有解，并且大于111111的奇数都可以写成999和大于444的偶数的和。所以只需要用x−9x-9x−9再按情况2计算即可。

综上所述，需要用到的数字只有4,6,94, 6, 94,6,9。

现在考虑分解出的数的个数。

  1. 当xxx是1,2,3,5,7,111, 2, 3, 5, 7, 111,2,3,5,7,11时，输出−1-1−1。
  2. 当xxx能被444整除时，输出x4\frac x4​4​​x​​。
  3. 当xxx是偶数且不能被444整除时，输出x−64+1\frac{x-6}4+1​4​​x−6​​+1。
  4. 当xxx是奇数时，输出x−94+1\frac{x-9}4+1​4​​x−9​​+1。

拓展

对于分解性的问题，如果需要最大化某个量，则极端地考虑可能的取值。

  • 如最大化数量，可以考虑让数字尽可能小。
  • 如最大化乘积，可以考虑让2,32, 32,3尽可能多。

然后再证明自己的猜想，如本题的"4,6,94, 6, 94,6,9"。
